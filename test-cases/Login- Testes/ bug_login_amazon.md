# Bug Report - Falha no login da Amazon

**ID:** BUG001  
**Resumo:** Login não funciona com senha correta  
**Passos para reproduzir:**
1. Acessar amazon.com.br
2. Inserir e-mail válido
3. Inserir senha correta
4. Clicar em "Fazer login"

**Resultado Esperado:** Acesso à conta do usuário  
**Resultado Obtido:** Erro "senha inválida" mesmo com senha correta  

**Severidade:** Alta  
**Prioridade:** Média  
**Ambiente:** Google Chrome 123.0, Windows 10
